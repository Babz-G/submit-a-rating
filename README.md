## 📌 Project Description & Purpose

An interactive rating component built with vanilla HTML, CSS, and JavaScript. Users can select a rating from 1 to 5 and submit it, which takes them to a thank you page that displays their selected rating.

## ✨ Features

- ⭐ Select a rating from 1 to 5
- ✅ Submit button to confirm the rating
- 🙏 Thank you page that displays the selected rating back to the user
